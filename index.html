PDF形式のパート譜のダウンロード・印刷方法

  1. 上の青色のダウンロードリンクからPDF形式のパート譜をダウンロードします。
  2. PDFファイルを開きます。PDFファイルは、お手持ちのPDFビュアー(Microsoft Edge、Adobe Acrobat DCなど)で表示・印刷できます。ここでは、Microsoft Edgeで表示した際の手順を説明します。
  3. 開くと、下の画像のような画面が表示されます。画像の赤枠の印刷アイコンをクリックして(あるいはctrl + Pで)印刷ダイアログを表示させます。
  4. 印刷設定を確認して、「印刷」ボタンを押し印刷を開始します。

オプション: パート譜に分かれているMuseScore Fileを入手する

下のリンクから、MuseScore File (*.mscz)を入手できます。これらのファイルはMuseScoreで直接表示・編集・印刷ができます。
